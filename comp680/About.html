How To Use:

  • Enter a starting / ending address. This program assumes you will be making a round-trip journey. An address can be any type of location that google maps can process, including: actual addresses with city, state, country, and zip code, geo-coordinates, intersections and more -- see: search locations on Google Maps.
  • Enter at least one location to visit in the first text area.
  • Click on the 'Add Another Address' button to add text areas that you can enter more locations to visit in. Note that you can only add locations once you have entered your start/end address and have entered a location in the first text box.
  • If you'd like to remove a location, simply click on the 'Remove Address' button. Note that this button is only for added locations.
  • Once your start/end address and at least one location have been entered, then click on the 'Submit' button to process a 'best route'. Note that the best route is distance between the location's geocoordinates as the crow flies, so your routing does not take into consideration actual roads or traffic on those roads.
  • Your 'best route' will be displayed in a new web page in Google Maps.
  • You may enter as many addresses as you want, but keep in mind that due to Google Maps geocoding constraints, for approximately every 10 addresses processed there will be a 3 second wait to avoid hitting Google's OVER_QUERY_LIMIT.

    • Error Messages:

  • You will see an error message if you attempt to add another address without first filling out the start/end and first locations fields.
  • You will see an error message if you submit the form without first filling out the start/end and first locations fields.
  • You will see an error message if you submit the form with locations that do not exist.
  • Should you encounter an error message, confirm that you have entered valid locations for the start/end address and the first location address. If any additional addresses have been entered then also confirm that those addresses are also valid.
